🐍 match_and_number_layers.py (15 KB)
   ⭐ НОВЫЙ! СОПОСТАВЛЕНИЕ СЛОЁВ
   
   Назначение:
   • Сопоставляет слои по геометрии между файлами
   • Восстанавливает правильную нумерацию
   • Инвариантен к смещению и вращению
   
   Использование:
   python match_and_number_layers.py simplified.svg optimized.svg
   
   Результат:
   optimized_numbered.svg с правильными номерами

🐍 process_svg_to_a4_pdf.py (17 KB)
   КОНВЕРТАЦИЯ SVG → PDF + РАЗДЕЛЕНИЕ НА A4
   
   Назначение:
   • Добавляет видимые номера на слои (опционально)
   • Конвертирует SVG в PDF (cairosvg)
   • Разделяет длинный PDF на страницы A4 (PyPDF2)
   
   Использование:
   python process_svg_to_a4_pdf.py input.svg
   
   Результат:
   input_A4.pdf готовый для печати
